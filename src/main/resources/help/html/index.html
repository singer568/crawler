二：架构说明
系统是基于 Swing+Spring-3.2.4+Mybatis-3.1.1+C3p0-0.9.1.2+Sqlite等技术实现的。
采用插件式开发模式，各大功能模块均可独出来，功能与系统UI对立，可配置性较强。

(1)我的丑牛：系统信息，插件信息,内存监控，以及任务监控。
